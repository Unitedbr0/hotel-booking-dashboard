# Hotel Booking Cancellation Dashboard

## Project Overview
This dashboard analyzes hotel booking and cancellation trends using Excel Pivot Tables, Charts, and Slicers.

## Features
- Total Bookings Analysis
- Cancellation Analysis
- Guest Type Analysis
- Room Type Analysis
- Monthly Booking Trends
- Interactive Slicers

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
